----- Exercice 1 -----

Information sur l'exercice 1

Confectionner un bouton permettant d'afficher une boîte de dialogue de type alert. Dedans, devra se trouver le message suivant afin d'être validé :
Félicitation vous avez réussi !

Pour compliquer le tout, il faudra faire en sorte de faire apparaître un popup avec bootstrap 4 avec bien entendu le résultat affiché dedans

{{ titreAlert }}

{{ messageAlert }}

{{ resultatAlert }}

----- Exercice 2 -----

Information sur l'exercice 2

- Créer un bouton permettant de changer le texte. De base le bouton affiche :
"Cliqué sur moi pour afficher le message secret"
- Automatiquement, une fois qu'on aura cliqué sur le message, le texte suivant apparaîtra permettant de valider partiellement l'exercice :
"Salut moussaillons, tu as réussi bravo !"

{{ messageSecret }}

A ce moment, le bouton affiche un tout autre message afin cette fois-ci de cacher le texte qui vient d'être affiché à l'écran. Le bouton affichera ainsi :
"Cachez vite ce message c'est très urgent."

----- Exercice 3 -----

Information sur l'exercice 3

Créer un bouton permettant d'afficher la date si possible au format DD/mm/YYYY et permettre qu'à chaque clic, de la mettre à jour. Afficher les minutes ainsi que les secondes afin de voir si l'exemple fonctionne bien

{{ maDate }}
